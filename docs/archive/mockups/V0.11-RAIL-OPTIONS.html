设计取舍

  • A · 无卡片药丸 — 最干净、跟其余 SalmonApp 暖白调一致、没有边框噪点。能看到 active 状态（药丸 + 左条 + salmon 字色）+ badge。我个人首选这个。
  • B · 暖暗色 — 跟内容区色差大，存在感强，永远不会问"图标栏在哪"。代价：深底跟 emoji 不太"友"，需要 desaturate。如果你想要更"专业工具"感选这个。
  • C · 圆形小头像 — 介于 A 和当前白卡之间。比 A 多点装饰感，比当前白卡少点边框噪点。
  • D · 图标+文字 — 学习成本最低，新用户看一眼就懂。代价：宽度多占 8px，6 个图标 + 2 个底部 = 8 行文字，密度增加。
你选哪个？ A / B / C / D，或者直接说"再改一下 X" 我继续微调。
选 A 也欢迎说"再去掉 active 的左条 / 把 badge 改小"等具体调整。
