(G) 首次添加账号 — Gmail / Outlook 二选一（v0.9 都做）

第一次进 📧 邮件 / 📅 日历 时弹出。OAuth 走系统浏览器（更安全且能用上你已登录的会话）。

添加邮箱账号
通过 OAuth 安全访问。refresh token 加密存在本机 keyring，邮件正文只发给你本地的 claude / codex CLI 做分析，不离开本机。
🔒 凭据加密存储 · 邮件正文不离开本机 · 你可以随时在 设置 → 账号 移除
  • 需授权 scope：Gmail (mail + drafts + send), Calendar (read/write), Contacts (read)
  • 同样的对 Outlook / Graph (Mail.ReadWrite, Mail.Send, Calendars.ReadWrite, Contacts.Read)

需要你拍板的（精简到 7 个）

  1. OAuth client ID：我帮你注册 SalmonApp 在 Google + Microsoft 各一个？还是 BYO？
  2. 首次同步范围：最近 90 天 + 1000 封邮件 起 OK 吗？太少 AI 上下文不够，太多首次同步慢。
  3. 富文本编辑器：tiptap（推荐 · 轻）vs lexical（重 · Meta 出）。
  4. AI token 消耗：每日 briefing 预估 50-100k tokens。要不要在 设置 → 用量 拆分"AI 自动消耗" vs "对话消耗"两列？
  5. 联系人 UI 入口：嵌在邮件视图里（推荐）vs 独立一级导航？
  6. 多账号同 provider：v0.9 是否支持加多个 Gmail / 多个 Outlook？还是每个 provider 限 1 个？
  7. 开发预算：5-6 个月内 v0.8.x CLI 线只接 hotfix，不接新功能。OK 吗？
